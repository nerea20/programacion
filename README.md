# Programación

Este repositorio tiene mis primeros ejercicios de programación en Java.
La idea es practicar conceptos básicos y compartir el progreso de lo que aprendo.

## Contenido

- **Ejercicios00** -> Esta carpeta contiene algunos ejemplo básicos.
- **Ejercicios1.2** -> Esta carpeta contiene ejercicios de uso de variables y E/S estándar.
- **Ejercicios1.3** -> Esta carpeta contiene ejercicios de cálculos simples.
- **Ejercicios2.6** -> Esta carpeta contiene ejercicios de condicionales.
- **Ejercicios2.7** -> Esta carpeta contiene más ejercicios de condicionales.
- **Ejercicios2.8** -> Esta carpeta contiene ejercicios de bucles.
- **Ejercicios2.9** -> Esta carpeta contiene más ejercicios de bucles.
- **Ejercicios3.10** -> Esta carpeta contiene ejercicios de vectores.
- **Ejercicios3.11** -> Esta carpeta contiene ejercicios de matrices.
- **Ejercicios3.12** -> Esta carpeta contiene ejercicios de cadena de caracteres.

### Ojetivo

- Practicar los fundamentos de Java
- Aprender a usar Git y GitHub
- Llevar un registro de mi progreso en programación
